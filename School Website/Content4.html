THE HOW MANY QUESTION

How many types of Malware exist ?


There are around 10 different kinds of Malware which affects our computer. Let us see them one by one.

1.Ransomware
Disables victim's access to data until ransom is paid.
Example: RYUK
2.Fileless Malware
Makes changes to files that are native to the OS.
Example: Astaroth
3.Spyware
Collects user activity data without their knowledge.
Example: DarkHotel
4.Adware
Serves unwanted advertisements.
Example: Fireball
5.Trojans
Disguises itself as desirable code.
Example: Emotet
6.Worms
Spreads through a network by replicating itself.
Example: Stuxnet
7.Rootkits
Gives hackers remote control of a victim's device.
Example: Zacinlo
8.Keyloggers
Monitors users' keystrokes.
Example: Olympic Vision
9.Bots
Launches a broad flood of attacks.
Example: Echobot
10.Mobile Malware
Infects mobile devices.
Example: Triada
